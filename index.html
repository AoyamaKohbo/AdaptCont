位置決め装置などのサーボ機構では駆動力の発生効率\(\eta\)にばらつきがあるため、 位置制御性能にばらつきが生じます。そこで、駆動力の発生効率\(\eta\)にばらつきがあっても、 常に一定の制御性能を実現するための適応制御について考察しましょう。 サーボ制御には比例微分制御が多く採用されるので、動的システムを次式とします。

\begin{align} \ddot x + \eta K_D \dot x + \eta K_P(x - u) = 0 \label{eq:SysEqn} \end{align}

ここに、\(x\)は位置、\(K_D\)は微分ゲイン、\(K_P\)は比例ゲイン、\(u\)は位置入力、\(\eta(>0)\)は効率係数です。 目的とする制御性能を規範モデルとして次式で与えます。

\begin{align} \ddot x_M + a\dot x_M + b(x_M - u)= 0 \label{eq:MdlEqn} \end{align}

ここに、\(x_M\)は規範モデルの位置です。 微分ゲイン\(K_D\)と比例ゲイン\(K_P\)は次式で調整可能とします。

\begin{align} K_D = \frac{a}{\hat\eta},~~~~K_P = \frac{b}{\hat\eta}. \label{eq:ModEta} \end{align}

ここに、\(\hat\eta\)は効率係数\(\eta\)の推定値です。 式\eqref{eq:ModEta}を式\eqref{eq:SysEqn}に代入して次式を得ます。

\begin{align} \ddot x + a\frac{\eta}{\hat\eta}\dot x + b\frac{\eta}{\hat\eta}(x - u) = 0 \label{eq:SysEqn2} \end{align}

式\eqref{eq:SysEqn2}において、\(\hat\eta=\eta\)であれば、規範モデル\eqref{eq:MdlEqn}と一致します。 すなわち、効率係数\(\eta\)のばらつきに関係なく目的とする制御性能が動的システムに対して実現できます。式\eqref{eq:SysEqn2}を次式のように変形します。

\begin{align} \ddot x + a \dot x + b (x - u) = (1- \frac{\eta}{\hat\eta})\{a \dot x + b (x - u) \} \label{eq:SysEqn3} \end{align}

式\eqref{eq:MdlEqn}の両辺から式\eqref{eq:SysEqn3}の両辺をそれぞれ引きます。

\begin{align*} (\ddot x_M - \ddot x) + a(\dot x_M - \dot x) + b(\hat b x_M - bx) = (1 - \frac{\eta}{\hat\eta})\{- a \dot x - b (x - u) \} \end{align*}

ここで、\(e = x_M - x\)、\(\tilde \eta = \hat\eta - \eta\)とおくと次式が得られます。

\begin{align} \ddot e + a e + b e = \frac{\tilde\eta}{\hat\eta} \{- a \dot x - b (x - bx)\} \label{eq:ErrEqn} \end{align}

Popovの超安定論を適用するために\(r\)と\(\varepsilon\)を次式で与えます。

\begin{align} r = \frac{\tilde\eta}{\hat\eta}\{- a \dot x - b (x - u)\},~~~\varepsilon = e + ce. \label{eq:ExdEqn} \end{align} 式\eqref{eq:ErrEqn}と\eqref{eq:ExdEqn} から線形前向きブロックの入力\(-r\)と出力\(\varepsilon\)の関係式は次式となります。 \begin{align} \ddot e + \hat a e + \hat b e = -r,~~~\varepsilon = e + ce. \label{eq:ErrEqn2} \end{align}

すると、Popov積分は次式となります。

\begin{align} \int_{0}^{t}r\varepsilon dt= \int_{0}^{t}\frac{\tilde\eta}{\hat\eta} \{- a \dot x - b (x - u)\}\varepsilon dt \label{eq:IntgEqn} \end{align}

適応制御を実現するために適応制御則を次式で与えます。

\begin{align} \dot{\tilde{\eta}} = \alpha \frac{\{- a \dot x - b (x - u)\}\varepsilon}{\hat\eta} \label{eq:AdptRule} \end{align}

ここに、\(\alpha\)は定数です。 式\eqref{eq:IntgEqn}に式\eqref{eq:AdptRule}を代入すると次式が得られます。

\begin{align*} \int_{0}^{t}r\varepsilon dt= \int_{0}^{t}\frac{1}{\alpha}\dot{\tilde\eta}\eta dt = [\frac{1}{2\alpha}\tilde\eta^2 ]_0^t \ge -\frac{1}{2\alpha}\tilde\eta(0)^2 \end{align*}

これで適応制御則\eqref{eq:AdptRule}によりPopovの積分不等式が成立することが確認できました。 \(\eta\)は定数なので、\(\dot{\tilde\eta} = \dot{\hat\eta}\)となることに注意すれば適応制御則は次式となります。

\begin{align} \dot{\tilde{\eta}} = \alpha \frac{\{- a \dot x - b (x - u)\}\varepsilon}{\hat\eta} \label{eq:AdptRule2} \end{align}

次に強正実条件を確認します。式\eqref{eq:ErrEqn2}の前向き線形ブロックの伝達関数\(G(s)=\varepsilon(s)/r(s)\)は次式となります。

\begin{align} G(s) = \frac{1+cs}{s^2+as+b} \label{eq:TransFanc} \end{align}

式\eqref{eq:TransFanc}の\(s\)に実数を代入すると\(G(s)\)も実数になり、s面の右半平面\(Re[s]\ge0\)に\(G(s)\)の極(分母：\(s^2+as+b=0\)の解)がないので、強正実条件(1)と(2)は確認できます。次に強正実条件(3)を確認します。

\begin{align} Re[G(j\omega)] = Re[\frac{1+jc\omega}{-\omega^2 +ja\omega + b}] > 0\label{eq:RealPart} \end{align}

分母を有利化すると次式になります。

\begin{align*} Re[G(j\omega)] &= Re[\frac{(1+jc\omega)(b-\omega^2 +ja\omega)}{(b -\omega^2)^2 + a^2\omega^2}]\\ &= Re[\frac{(b-\omega^2+ac\omega^2)+j\{c\omega(b-\omega^2)-a\omega\}}{(b -\omega^2)^2 + a^2\omega^2}]\\ &= \frac{b+(ac-1)\omega^2}{(b -\omega^2)^2 + a^2\omega^2}> 0 \label{eq:RealPart2} \end{align*}

分子の中の\(ac-1>0\)、すなわち\(c>1/a\)であれば、強正実条件(3)が満足されます。

動的システム\eqref{eq:SysEqn}を規範モデル\eqref{eq:MdlEqn}に適応させる適応制御則と強正実条件を以下にまとめます。

適応制御則：

\begin{align*} \dot{\hat{\eta}} = \alpha \frac{\{- a \dot x - b (x - u)\}\varepsilon}{\hat\eta} \end{align*}

強誠実条件：

\begin{align*} c > \frac{1}{a} \end{align*}
